# 📊 Globussoft Data Science Assignment

This repository contains the implementation of both tasks as part of the Data Science assignment.

---

## 🚀 Tasks Completed

### 🔹 Task 1: Amazon Product Scraper
- Scrapes laptop data from Amazon India
- Extracts:
  - Product Title
  - Price
  - Rating
  - Image URL
  - Ad / Organic Result
- Saves output in a **timestamped CSV file**

---

### 🔹 Task 2: Drone Count Detection (FastAPI)
- Built a FastAPI service for detecting drones in a sky image
- Uses a lightweight **YOLOv8 model**
- Returns:
  - Bounding boxes
  - Confidence scores
  - Total count of detected drones.
